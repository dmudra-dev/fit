Strava
API Auto-Sync
Enter your Strava access token once and sync activities on demand — no CSV, no email, no archive download.
One-time setup (~3 min):
1. Go to strava.com/settings/api and create an app (any name)
2. Visit: https://www.strava.com/oauth/authorize?client_id=YOUR_ID&response_type=code&redirect_uri=http://localhost&approval_prompt=force&scope=activity:read_all
3. Approve → copy the code= value from the redirect URL
4. Exchange for a token via strava.com/oauth/token (POST) — paste result below
Or use a Strava token generator tool (search "Strava token generator") for a simpler flow.
